# App Empleos Informales
Aplicación móvil desarrollada en Apache Cordova que tiene como fin ofrecer a las personas ofrecer sus servicios profesionales para ser contratados, así como buscar profesionales.
El aplicativo tiene como objetivo abrir una brecha de oportunidades a los oficios informales tales como carpinteros, pintores, albañiles, mecánicos, etc., gratuitamente. La idea es que aquellos que tienen ciertas habilidades puedan publicarse en la aplicación y de la misma forma aquellos que necesiten sus servicios pueden contratarlos, así como calificarlos para que otros puedan confiar en la calidad del trabajo realizado.

Nace de la primicia mucho talento y no hay oportunidades en el campo informal. 
Cuántas veces hemos querido contratar a alguien para pintas la casa? Cuántas veces hemos necesitado un mecánico? y muchas veces hasta el vecino podría tener este talento pero no existe el medio para publicarse y contratar.

## Detalles Técnicos 

- Framework de desarrollo, Apache Cordova.
- API Rest en PHP.
- Diseño y acabados con Bootstrap (html, css y JavaScript).
